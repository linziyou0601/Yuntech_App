# YunTech App
　　本App為非官方軟體，為方便使用手機查詢資料而開發，內容僅供查詢參考，所有資訊皆以「雲科大單一入口網」為準。此外，本App運作時會將帳號密碼加密處理，且不會主動紀錄使用者之帳號密碼，若有疑慮請勿使用。

## 功能
- 個人資訊
- 課表查詢
- 學期成績
- 應修未修畢業學分
- 校園地圖
- 最新消息
- 焦點消息
- 課程查詢
